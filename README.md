# 🎧 Simple VC Music Bot

Telegram Voice Chat Music Bot using Pyrogram v2 + PyTgCalls

## 🚀 Deploy

### Heroku
Click Deploy button and fill env vars.

### Railway
Deploy from GitHub → add variables → done

### VPS

git clone your_repo  
cd simple-vc-music-bot  
pip install -r requirements.txt  
python bot.py

## ⚙️ Commands

/play song name  
/stop

## ⚠️ Requirements

- Assistant account in group
- Admin with VC permission
- Voice chat started
